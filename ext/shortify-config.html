Shortify

Transforms ch[story-id] mentions to Shortcut links.

Changes are saved automatically.

Settings

Pages
Optional, workspace to use in the links. e.g.: https://app.shortcut.com/your-workspace-name/... If empty, Shortcut will redirect you to the right workspace, but you can avoid this extra redirect by specifying it here.
Comma-separated, can be any portion of the url. For instance, to match any git server (github, gitlab, etc.), use "git". To restrict it to pull requests, use "git.example.com/myorg/myproject/pull". Regexes NOT currently supported.
Comma-separated elements that you would like to ignore. If you know you won't need to Shortify any links in a particular element this could speed up the process. Example: "textarea,code,div"
